# Napoletana Nostra Landing Page

A beautifully designed, premium website landing page concept for an authentic Neapolitan eatery/brand, built using clean HTML and modern semantic CSS.

## ✨ Features
* **Premium Aesthetic:** Features a custom color palette including Ivory, Gold, Terra Cotta, and Olive.
* **SVG Grain Overlay:** Uses an embedded SVG filter to give the background a subtle, rustic paper/grain texture.
* **Fluid Navigation:** Includes a fixed, blur-effect (`backdrop-filter`) navigation bar.
* **Typography:** Integrates elegant Google Fonts (`Cormorant Garamond`, `Libre Baskerville`, and `Jost`).
* **Fully Responsive:** Optimizes beautifully for mobile, tablet, and desktop viewports.

## 🚀 Quick Start
Simply clone the repository and open the `index.html` file in any modern web browser.
